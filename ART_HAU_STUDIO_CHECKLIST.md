## 🏗️ PHASE 0: Chuẩn bị cơ sở vật chất (Tháng 7/2026)

### Không gian
- [ ] Đo đạc không gian 3 khu: Studio A (60m²), Xưởng B (80m²), Gallery C (100m²)
- [ ] Lắp vách kính ngăn phòng (35tr)
- [ ] Lắp điều hòa — Phòng học + Xưởng (30tr)
- [ ] Sơn sửa tường, trần (40tr)
- [ ] Lắp hệ thống đèn gallery (treo tranh, chiếu điểm)
- [ ] Lắp hệ thống rửa cho Xưởng B (khu rửa cọ, đất sét)
- [ ] Kê bàn ghế Studio A (20 bộ)
- [ ] Kê bàn nặn + kệ Xưởng B (6 bàn)
- [ ] Lắp hệ thống treo tranh + giá trưng bày di động Gallery C
- [ ] Lắp âm thanh + máy chiếu Gallery C

### Trang thiết bị
- [ ] Mua 25 giá vẽ chuyên dụng
- [ ] Mua 20 bộ màu acrylic cao cấp
- [ ] Mua 100 khung vải canvas
- [ ] Mua 2 máy in lụa thủ công
- [ ] Mua máy chiếu + màn chiếu
- [ ] Mua hệ thống âm thanh
- [ ] Mua bàn nặn gốm (6 bàn)
- [ ] Mua giá trưng bày di động (10 cái)
- [ ] Mua vật tư tiêu hao tháng đầu (giấy, sơn, bút, đất sét)

### Vật tư tồn kho ban đầu (cho 3 tháng)
- [ ] Canvas các khổ (20×20, 30×40, 50×60)
- [ ] Acrylic: đen, trắng, 6 màu cơ bản, 4 màu đặc biệt
- [ ] Màu nước: 10 bộ student grade
- [ ] Sơn dầu: 5 bộ starter
- [ ] Đất sét Nhật: 20kg
- [ ] Bút vẽ: các cỡ 0-12 (50 cây)
- [ ] Giấy vẽ A3, A4: 500 tờ
- [ ] Tạp dề: 30 cái
- [ ] Bảng pha màu: 20 cái
