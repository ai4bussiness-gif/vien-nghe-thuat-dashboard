## 3. TRIỂN LÃM — TALKSHOW

### 3.1. Quy trình 1 triển lãm (6-8 tuần chuẩn bị)

| Giai đoạn | Việc | Thời gian |
|-----------|------|:---------:|
| **Concept** | Xác định chủ đề, curator, nghệ sĩ tham gia | T-8 tuần |
| **Chọn tác phẩm** | Gửi lời mời nghệ sĩ, nhận tác phẩm, chọn lọc | T-6 tuần |
| **Sản xuất** | In ấn, đóng khung, viết giới thiệu, làm catalogue | T-4 tuần |
| **Truyền thông** | Đăng teaser, poster, gửi media invite | T-2 tuần |
| **Setup** | Treo tranh, sắp đặt, test ánh sáng + âm thanh | T-2 ngày |
| **Khai mạc** | Đón khách, khai mạc, networking, chụp ảnh | Ngày X |
| **Vận hành** | Mở cửa cho công chúng (7-14 ngày), có người trực | X+1 → X+14 |
| **Kết thúc** | Hạ tác phẩm, trả lại nghệ sĩ, tổng kết | X+14 |

### 3.2. Talkshow — quy trình nhanh hơn (2-3 tuần)

```
1. Chọn chủ đề + diễn giả (3 tuần trước)
2. Đăng bài giới thiệu + mở đăng ký (2 tuần)
3. Setup phòng: micro, ghế ngồi, nước, màn chiếu (H-1 ngày)
4. Talkshow: 60-90 phút (talk 45' + Q&A 30')
5. Recap: bài tổng kết + ảnh + video highlight
```

## 4. CUỘC THI

### 4.1. Quy trình tổ chức 1 cuộc thi

```
─── T-8T ──→ T-6T ──→ T-4T ──→ T-2T ──→ T-1T ──→ NGÀY THI ──→ T+2T
Xác nhận     Mở đơn    Hết hạn   BGK       Setup     Vòng chung   Công bố
thể lệ       + tìm     nhận bài   chấm      vòng 2    kết +        + gửi
+ BGK        tài trợ   vòng 1    vòng 1    + in ấn   trao giải    tác phẩm
                                                   + truyền thông  trưng bày
```

### 4.2. Checklist cuộc thi

| Việc | Ghi chú |
|------|---------|
| Soạn thể lệ + quy chế | Gửi Viện duyệt |
| Tìm nhà tài trợ | Doanh nghiệp sơn, giấy, họa cụ |
| Thiết kế poster + banner | Ấn phẩm truyền thông |
| Mở đơn + nhận bài | Online form |
| Truyền thông liên tục | 1 bài/tuần + teaser giải thưởng |
| Hết hạn nhận bài → chấm vòng 1 | BGK (3-5 người) |
| Công bố top 10/20 vào vòng chung kết | PR mạnh trước chung kết |
| Setup không gian chung kết | Gallery + sân khấu |
| Tổ chức chung kết + trao giải | Nên kết hợp khai mạc triển lãm |
| Trưng bày tác phẩm đoạt giải | 1-2 tuần sau chung kết |
| Cảm ơn nhà tài trợ + tổng kết | Báo cáo + gửi đối tác |
