# Dashboard Viện Nghệ Thuật

## Tech stack
- Astro + TypeScript + Tailwind CSS
- Supabase (PostgreSQL) for data
- Deploy: Cloudflare Pages

## Project commands
- `npm run dev` — dev server
- `npm run build` — build
- `npm run preview` — preview build

## Structure
- Pages: src/pages/
- Components: src/components/
- Layout: src/layouts/
