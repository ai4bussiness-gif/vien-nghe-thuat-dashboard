# Dashboard Viện Nghệ Thuật

## Tech stack
- Astro (v6) + TypeScript + Tailwind CSS v4
- static output (no server adapter)
- Deploy: Cloudflare Worker (static assets)

## Project commands
- `npm run dev` — dev server
- `npm run build` — build
- `npm run preview` — preview build
- `npx wrangler deploy` — deploy to Cloudflare

## Structure
- Pages: src/pages/
- Components: src/components/
- Data: src/data/
- Layout: src/layouts/

## Rules
Đọc file RULES.md để biết đầy đủ quy tắc đặt tên, style, và debug.
